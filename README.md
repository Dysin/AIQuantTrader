### 1️⃣ 宏观信息（Macro-Level Factors）

宏观环境是股市的“温度计”，它决定了市场的总体趋势和资金流向。

| 经济周期阶段              | 特征                  | 对股市的影响             | 投资策略                   |
| ------------------- | ------------------- | ------------------ | ---------------------- |
| **复苏期（Recovery）**   | GDP增速开始回升，利率偏低，通胀温和 | 企业盈利改善，股市底部形成，上涨初期 | 抄底优质蓝筹与周期股（如制造业、金融、地产） |
| **繁荣期（Expansion）**  | 经济高增长，企业利润大增，流动性充裕  | 股市普遍上涨，估值上升，部分泡沫出现 | 持有成长股，警惕高估，逐步减仓        |
| **衰退期（Recession）**  | 经济增速放缓，企业利润下滑，需求减弱  | 股市高位回落，资金流出风险资产    | 防守为主，持有现金、黄金或高股息股      |
| **萧条期（Depression）** | 经济负增长，失业率高，政策刺激出台   | 市场筑底阶段，估值极低        | 逐步建仓，关注政策受益行业（基建、新能源等） |

#### ➕ 其他关键宏观指标：

* **利率政策**：加息 → 资金成本上升 → 股市承压；降息 → 资金流入股市 → 利好股市。
* **通货膨胀率**：适度通胀（2%~3%）有利企业盈利；高通胀 → 压缩消费、推高利率 → 利空股市。
* **财政政策**：扩张性（减税、基建投资） → 提振经济、利好股市；紧缩性政策 → 抑制增长。
* **地缘政治与国际贸易**：贸易摩擦、战争 → 增加市场不确定性 → 资金避险（黄金、债券受益）。

### 中国宏观杠杆率

**定义与计算方式**  
中国宏观杠杆率（Macro Leverage Ratio）是衡量整个国家债务水平的重要指标，反映经济运行中债务扩张对GDP的支撑程度。  
计算公式为：  
> 宏观杠杆率 = （政府部门债务 + 非金融企业债务 + 居民部门债务） ÷ GDP × 100%

该指标由中国人民银行与国家金融与发展实验室（NFDS）联合发布，通常按季度公布。

**经济含义**  
宏观杠杆率代表一个经济体“用债务撬动经济增长”的程度，是金融稳定与经济可持续发展的核心参考指标：  
- **上升阶段**：意味着经济依赖信贷扩张推动增长；  
- **稳定阶段**：说明债务与产出保持协调；  
- **过高或快速上升**：则暗示潜在的金融风险与债务泡沫。  

**主要构成**  

| 部门 | 含义 | 特征 |
|------|------|------|
| 政府部门 | 中央与地方财政债务 | 稳定增长，受基建与财政政策影响 |
| 非金融企业部门 | 企业贷款与债券融资 | 历史最高，占比最大 |
| 居民部门 | 房贷、消费贷等 | 房地产周期影响显著 |

**数据来源**  
- 国家金融与发展实验室季度报告  
- 中国人民银行金融统计数据  
- 国家统计局《国民经济和社会发展公报》

**对股市的影响**  
- 📈 **短期影响**：杠杆率上升通常伴随信贷扩张、流动性宽松，对股市形成资金面支撑；  
- 📊 **中期影响**：如果经济增长同步改善（投资回报率 > 融资成本），股市往往处于上升周期；  
- 📉 **长期影响**：若债务效率下降（新增债务对GDP拉动减弱），则预示经济潜在风险积累，股市可能进入震荡或下行阶段。

**投资启示**  
- 杠杆率持续上升但经济数据疲弱 → 防范系统性风险；  
- 杠杆率温和增长且政策支持明显 → 股市通常进入估值修复期；  
- 重点关注**结构性杠杆变化**：企业去杠杆+政府扩杠杆往往对应“稳增长+结构调整”行情。

### 国民经济运行指数（Economic Prosperity Index）

**定义与构成**  
国民经济运行指数（或称“景气指数”）是反映宏观经济整体活跃度的综合指标，由工业生产、投资、消费、进出口、就业、价格等多个数据加权计算而成。  
> 指数 = 100 为经济运行正常水平  
> 指数 > 100 表示扩张  
> 指数 < 100 表示收缩  

**经济含义**  
该指数是衡量经济周期阶段的重要工具，可直观反映经济是否处于“复苏”、“过热”或“下行”阶段。  
它与GDP增速和工业增加值增长率高度相关。

**数据来源**  
- 国家统计局经济景气监测中心  
- 《国民经济运行月度报告》  
- 各地统计局月度经济运行快报  

**历史趋势（参考近年）**  
- 疫情前：指数长期处于 101~103（温和扩张）；  
- 2020年疫情冲击：降至 95 以下；  
- 2021-2024年：逐步修复，保持在 99~101 之间波动，显示经济处于“弱复苏”阶段。

**对股市的影响**  
- 📈 **短期**：指数上升通常对应制造业扩张、企业盈利改善，对A股形成正面推动；  
- 📊 **中期**：指数连续三个月上行常被视为经济复苏信号，带动周期股和金融板块；  
- 📉 **长期**：若指数高位回落，往往意味着经济过热后放缓，股市可能进入调整期。  

**投资启示**  
- 经济运行指数是判断市场风格切换的重要信号；  
- 扩张期优先配置**周期性行业（钢铁、化工、金融）**；  
- 收缩期转向**防御型板块（消费、公用事业）**。

### 外商直接投资数据（FDI - Foreign Direct Investment）

**定义**  
外商直接投资（FDI）是指外国企业或个人以长期利益为目的，在中国设立企业、并购企业或扩张投资的行为所形成的资本流入。

**计算方式**  
> FDI = 当期外资企业新设与再投资总额（含股权投资、债务投资、留存收益再投资）

**经济含义**  
FDI反映外资对中国经济、制度环境、产业前景的信心，是衡量国际资本对中国长期预期的重要指标。  
- 高FDI：说明中国投资环境吸引力强；  
- FDI下降：可能反映全球风险偏好下降或外资观望情绪。  

**数据来源**  
- 商务部（MOFCOM）月度统计公告  
- 国家外汇管理局（SAFE）  
- 联合国贸发会议（UNCTAD）《世界投资报告》

**历史趋势（截至2024年）**  
- 2015-2019年：稳步增长，FDI约1200亿美元/年；  
- 2020疫情后：短期下滑，随后恢复；  
- 2023-2024年：结构性变化明显，高技术制造业与新能源领域吸引更多外资。  

**对股市的影响**  
- 📈 **短期**：FDI流入提升人民币汇率与市场信心，推动外资加仓A股；  
- 📊 **中期**：外资持续投资制造业，有助于提高相关上市公司盈利预期；  
- 📉 **长期**：若全球资本环境趋紧（如美联储加息周期），FDI放缓将压制股市估值修复。

**投资启示**  
- 重点关注外资流向：高技术制造、绿色能源、数字经济；  
- 若FDI结构优化（高附加值产业占比提升），长期利好A股产业升级逻辑；  
- FDI下降不一定是利空，若伴随出口回升与内需扩张，则市场仍有支撑。

---

### 外商直接投资增长率（FDI Growth Rate）

**定义与计算公式**  
外商直接投资增长率表示外资进入中国的速度变化，是衡量外资信心变化的领先指标。  
> FDI增长率 = （本期FDI - 上期FDI） ÷ 上期FDI × 100%

**经济含义**  
该指标比FDI总量更具“趋势性”，能提前反映资本流入的加速或放缓。  
- 增长率上升 → 外资信心增强；  
- 增长率下降 → 外资谨慎或撤离。  

**数据来源**  
- 商务部《全国吸收外资情况报告》  
- SAFE国际收支统计  
- UNCTAD季度全球FDI流量报告  

**历史趋势（截至2024年）**  
- 2016-2019年：增速保持在5%~10%；  
- 2020年疫情冲击后：短期下滑约-15%；  
- 2021-2023年：恢复正增长，尤其新能源与高端制造业吸引外资显著；  
- 2024年：受全球资本紧缩影响，增速趋缓但结构优化。  

**对股市的影响**  
- 📈 **短期**：增长率上升通常利好A股、港股外资重仓板块（消费、医药、制造）；  
- 📊 **中期**：持续正增长意味着外资回流、市场估值修复；  
- 📉 **长期**：若增长率持续为负，可能触发资金外流与风险偏好下降，抑制股市反弹。

**投资启示**  
- 观察FDI增长率变化可提前判断外资情绪；  
- 当FDI增速转正时，是A股外资配置加速期；  
- FDI增速与人民币汇率、沪深港通资金流向高度相关，可作为市场中期配置信号。

### LPR品种数据（Loan Prime Rate）

**定义与背景**  
贷款市场报价利率（Loan Prime Rate，简称 **LPR**）是中国银行业对优质客户发放贷款的基准利率，由 **全国银行间同业拆借中心（NIFC）** 根据18家报价行报出的利率加权计算得出。  
LPR改革自 **2019年8月** 起正式实施，取代了原贷款基准利率体系，成为中国货币政策传导的重要工具。

LPR分为两个主要品种：
- **1年期LPR**：反映短期融资成本，主要影响企业生产经营贷款、短期信用贷款。  
- **5年期以上LPR**：反映长期融资成本，主要影响房贷利率与基建投资成本。

**计算方式**  
> LPR = 报价行参考中国人民银行中期借贷便利（MLF）利率 + 报价行加点（反映风险溢价）

每月 **20日（遇节假日顺延）** 公布最新LPR利率，单位为年利率（%）。

**经济含义**  
LPR是中国货币政策调控体系中的核心利率之一，具有以下意义：
1. 体现银行资金成本与信贷供需变化；  
2. 作为企业与居民贷款利率的定价基准；  
3. 反映货币政策松紧方向。  

LPR下降 → 货币政策宽松 → 降低企业与居民融资成本；  
LPR上升 → 货币政策收紧 → 抑制信贷扩张与通胀。

**数据来源**  
- 中国人民银行（PBoC）官方网站  
- 全国银行间同业拆借中心（NIFC）  
- Wind、东方财富Choice、CEIC数据库  

**历史趋势（截至2024年）**  

| 年份 | 1年期LPR | 5年期LPR | 政策取向 |
|------|-----------|-----------|-----------|
| 2019年 | 4.25% | 4.85% | 改革启动，利率市场化 |
| 2020年 | 3.85% | 4.65% | 疫情宽松周期 |
| 2021年 | 3.85% | 4.65% | 稳定期 |
| 2022年 | 3.65% | 4.30% | 结构性宽松 |
| 2023年 | 3.45% | 4.20% | 房贷利率下调、刺激需求 |
| 2024年 | 3.35% | 4.00% | 宽货币周期延续 |

趋势特征：  
- 总体呈 **下行趋势**，反映宽松货币政策环境；  
- 1年期LPR与MLF利率变动高度同步；  
- 5年期LPR更直接影响房地产与长期信贷。

**对股市的影响**  

| 时间尺度 | 影响机制 | 股市表现 |
|------------|------------|------------|
| 📈 **短期** | LPR下调 → 降低企业融资成本 → 提振风险偏好 | 股市利好，尤其是地产、金融、消费板块 |
| 📊 **中期** | LPR下降带动信贷扩张与投资增长 | 有助于经济复苏，提升企业盈利预期 |
| 📉 **长期** | 若长期低利率叠加经济疲软 → 流动性陷阱风险 | 股市表现分化，成长股相对占优 |

**结构性影响**  
- **1年期LPR** 影响短期贷款与企业融资 → 制造业、周期股敏感；  
- **5年期LPR** 影响按揭贷款与地产投资 → 房地产、银行板块高度相关；  
- LPR下调周期往往对应**货币宽松、股市估值修复阶段**。

**投资启示**  
- 关注 LPR 与 **MLF利率** 之间的传导关系：MLF下降通常预示LPR下调；  
- LPR下调初期，利好**高负债行业（地产、基建）**与**高估值板块（成长股）**；  
- 若LPR连续下调但经济指标疲弱，需警惕“流动性陷阱”行情（即股市反应有限）。

---

### LPR与其他宏观指标的联动关系

| 指标 | 相关性 | 联动逻辑 |
|------|----------|-----------|
| **M2增速** | 正相关 | M2扩张常伴随LPR下降（宽货币） |
| **社会融资规模** | 正相关 | LPR下调促进信贷扩张与融资需求 |
| **CPI/PPI** | 反相关 | LPR下降通常出现在通胀下行期 |
| **人民币汇率** | 弱负相关 | LPR下调可能导致人民币短期走弱 |
| **A股估值（PE）** | 正相关 | 低利率环境提升股市估值中枢 |

---

**总结**
> LPR是中国货币政策的“晴雨表”。  
> 1年期反映短期融资成本；5年期影响长期资金价格。  
> LPR下调通常代表宽松周期开启，对股市资金面形成支撑；  
> 但若伴随经济下行压力，则股市上涨可持续性需结合基本面判断。

### 居民消费价格指数（CPI, Consumer Price Index）

**定义与计算方式**  
居民消费价格指数（CPI）是反映居民家庭购买的生活消费品和服务项目价格变动情况的综合指数，用以衡量通货膨胀或通货紧缩的程度。  

计算公式：  
> CPI = （当期消费支出价格水平 ÷ 基期消费支出价格水平） × 100%

即以一个基准年份（如2020年=100）为参照，比较各期居民消费品和服务价格的总体变化。  

CPI的分类一般包括：  
1. 食品类  
2. 居住类（含房租、水电燃料等）  
3. 交通通信类  
4. 教育文化娱乐类  
5. 医疗保健类  
6. 其他用品和服务  

**核心子指标**  
- **CPI同比（YoY）**：反映价格较上年同期的变化幅度，是通胀的主指标。  
- **CPI环比（MoM）**：反映当月较上月的价格变动，是短期价格走势的重要参考。  
- **核心CPI**：剔除食品和能源等波动较大的项目，更能反映中长期价格趋势。  

---

**经济含义**  
CPI是宏观经济分析中最具代表性的“物价指标”，反映居民生活成本与整体物价水平变化，对货币政策和市场预期影响重大：  
- CPI上升 → 通货膨胀加剧 → 货币政策趋紧；  
- CPI下降 → 通货紧缩风险 → 货币政策宽松。  

稳定的CPI（约2%上下）通常被视为经济健康增长的表现。  

---

**数据来源**  
- 国家统计局（NBS）《居民消费价格指数月度报告》  
- Wind / CEIC / 东方财富Choice 数据库  
- 中国人民银行统计数据（用于货币政策分析）  

发布频率：**每月发布上月数据**（通常在次月10日左右公布）。

---

**历史趋势（截至2024年）**

| 年份 | CPI同比（全年均值） | 核心CPI | 政策解读 |
|------|----------------------|----------|-----------|
| 2019年 | 2.9% | 1.6% | 猪肉价格推高通胀 |
| 2020年 | 2.5% | 0.8% | 疫情影响消费需求下降 |
| 2021年 | 0.9% | 0.8% | 经济修复、需求偏弱 |
| 2022年 | 2.0% | 0.9% | 温和通胀 |
| 2023年 | 0.2% | 0.7% | 通缩压力显现 |
| 2024年 | 0.4% | 0.8% | 消费温和回升，整体低通胀 |

> 趋势说明：  
> 过去数年CPI维持在低位区间，说明中国经济处于**低通胀—宽货币—稳增长**的宏观环境。

---

**对股市的影响**

| 时间尺度 | 主要影响机制 | 股市影响特征 |
|------------|----------------|----------------|
| 📈 **短期** | CPI上升 → 物价上涨带动企业收入增加，但若成本上升更快则压缩利润 | 对消费类股偏利好，对制造业偏中性或负面 |
| 📊 **中期** | CPI连续上升 → 通胀预期上升，央行或收紧货币政策 | 流动性收紧、利率上升，对股市估值不利 |
| 📉 **长期** | CPI持续低位或为负 → 反映需求疲软，央行或实施降息、降准 | 资金面宽松，股市估值修复、成长股受益 |

---

**行业敏感性分析**

| 行业 | CPI上升影响 | CPI下降影响 |
|-------|----------------|----------------|
| 消费品（食品、饮料） | 收入改善、利润提升 | 销量可能下滑 |
| 制造业 | 成本压力上升 | 原材料成本下降、利润修复 |
| 银行与保险 | 加息周期盈利提升 | 降息周期息差收窄 |
| 房地产 | 高CPI易导致加息、抑制房市 | 低CPI利率下降、房贷成本降低 |
| 公用事业 | 受通胀影响较小 | 稳定板块、通缩期防御性强 |

---

**CPI与其他宏观指标的联动关系**

| 指标 | 相关性 | 联动逻辑 |
|------|----------|-----------|
| **PPI（工业品出厂价格指数）** | 领先指标 | PPI上涨通常先于CPI上行；PPI持续低迷预示CPI通缩风险 |
| **M2增速** | 弱正相关 | 货币投放增加会在滞后1-2个季度推动CPI上行 |
| **LPR利率** | 反相关 | CPI下行常伴随LPR下降（宽松周期） |
| **社会融资规模** | 正相关 | 信贷扩张有助于提升消费与通胀预期 |
| **汇率（人民币兑美元）** | 弱负相关 | 人民币贬值推高输入型通胀 |

---

**投资启示**

1. **CPI快速上升阶段（通胀周期）**  
   - 关注：食品、农产品、能源、资源品板块  
   - 风险：政策收紧、利率上行、成长股估值受压  

2. **CPI回落或为负阶段（通缩周期）**  
   - 关注：消费、科技、成长、医药板块  
   - 逻辑：流动性宽松、政策刺激力度增强  

3. **CPI稳定阶段（温和通胀 1~3%）**  
   - 是最有利于股市中长期上涨的宏观环境  
   - 投资与盈利预期稳定、风险偏好较高  

---

**总结**
> CPI是宏观经济分析的“温度计”。  
> 当CPI高企时，意味着经济“过热”需降温；当CPI偏低时，说明需求不足，央行可能放松政策。  
> 对股市而言，**温和通胀最有利、通缩次之、高通胀最不利**。  
> 结合CPI走势与政策取向，可判断货币环境与资产配置方向。

### 费城半导体指数（Philadelphia Semiconductor Index, SOX）

**定义**  
费城半导体指数（SOX，Philadelphia Semiconductor Index）由美国费城证券交易所于 1993 年推出，是反映全球半导体行业整体表现的权威指数。  
该指数以在美国上市的主要半导体公司为样本股，包括**芯片设计、制造、封测、设备与材料**等全产业链企业。  
其成分股涵盖行业龙头，如：英伟达（NVIDIA）、台积电（TSMC ADR）、英特尔（Intel）、博通（Broadcom）、美光（Micron）等。

**计算方式**  
SOX 为市值加权指数，样本企业权重随市值动态调整。  
指数基期为 1993 年 12 月 1 日，基点为 200 点。  
> 当前指数水平反映自基期以来半导体板块整体市值变化倍数。

**经济含义**  
费城半导体指数被视为**全球科技周期与资本市场风险偏好**的晴雨表：  
- 半导体是现代工业与科技创新的核心基础，产业链广泛影响消费电子、汽车、人工智能、云计算、通信等领域。  
- 指数的涨跌通常反映全球科技需求、企业资本支出周期及技术创新热度。  
- 因半导体行业高度周期性，SOX走势往往领先于全球经济与股市周期 6-12 个月。

**数据来源**  
- 费城证券交易所（NASDAQ PHLX）  
- 纳斯达克官网  
- 彭博（Bloomberg）、路孚特（Refinitiv）、TradingView 等财经终端  
- Yahoo Finance 指数代码：`^SOX`

**对股市的影响**

| 时间维度 | 影响方向 | 说明 |
|-----------|-----------|------|
| 📈 **短期（1-3个月）** | 正相关 | SOX 上涨代表市场对科技与创新预期乐观，风险偏好提升，A股中半导体、芯片、AI概念股通常联动上涨。 |
| ⚖️ **中期（3-6个月）** | 领先性 | SOX 指数回调往往早于全球科技股见顶，可作为中国科创板、创业板等板块的领先信号。 |
| 📉 **长期（6-12个月）** | 宏观联动 | 若SOX持续下跌，通常反映全球电子需求下行、资本开支收缩，科技板块估值面临压力。 |

**延伸解读**  
- 半导体行业景气与全球制造业 PMI、DRAM/NAND 价格、晶圆厂设备订单高度相关。  
- 费城半导体指数与**纳斯达克指数（NASDAQ）**的相关系数长期超过 0.8，是判断科技牛市或回调的重要指标。  
- 中国市场受影响较大的板块包括：
  - 沪深半导体产业链（设备、设计、材料）
  - 科创板高端制造与AI相关概念股
  - 人工智能、算力基础设施与新能源车电子类公司  

**典型市场反应案例**
- ✅ **2020-2021年**：SOX 指数随全球AI与5G浪潮暴涨，中国A股半导体板块同步进入高估值阶段。  
- ⚠️ **2022年**：美联储加息周期启动，SOX 回调超过 40%，中国芯片股同步大幅下跌。  
- ✅ **2023下半年起**：NVIDIA 业绩爆发、AI需求上升，SOX 再创新高，引发全球科技股反弹。  

**结论**  
费城半导体指数不仅是全球半导体产业的温度计，也是**中国A股、港股科技板块的先行指标**。  
研究其走势可辅助判断：
- 全球科技投资周期阶段  
- 资金风险偏好变化趋势  
- 中国科技股中短期机会与风险点  

### 原油运输指数（Baltic Dirty Tanker Index, BDTI）

**定义**  
原油运输指数（BDTI）由波罗的海交易所（Baltic Exchange）编制，反映全球原油海运市场的运价水平，是国际航运业中最具代表性的**原油船运价指数**之一。  
该指数综合统计多条主要航线（如中东至日本、西非至美国、北海至地中海等）的原油运输现货运价，按权重加权平均得到每日指数值。

**计算方式**  
> BDTI = 各航线加权平均运价（以世界尺度航程和吨位为权重）

指数单位为“点”，基期设为 1985 年 1 月 4 日，基点为 1000。  
指数随国际原油运输市场即期运价的波动而变动。

**经济含义**  
BDTI 反映全球原油海上运输供需状况，是国际贸易与能源市场的重要前瞻性指标。  
- 当指数上升 → 说明油轮需求增加、运价上涨，往往与**原油贸易活跃、全球经济复苏**相关；  
- 当指数下行 → 表示运输需求疲软，可能预示**全球需求下滑或原油库存高企**。  

**数据来源**  
- 波罗的海交易所（Baltic Exchange）  
- Refinitiv、Bloomberg、TradingView、英为财情（Investing.com）  
- 航运咨询机构 Clarksons、Baltic Briefing  

**对股市的影响**

| 时间维度 | 影响方向 | 解读 |
|-----------|-----------|------|
| 📈 **短期（1-3个月）** | 正相关 | BDTI 上升意味着全球原油运输需求上升，能源、航运、造船等板块普遍受益。 |
| ⚖️ **中期（3-6个月）** | 宏观信号 | 持续高位运行往往预示全球经济活动回暖，带动周期股上涨。 |
| 📉 **长期（6-12个月）** | 通胀预警 | 若指数长期高企，运输成本上升会推高原油终端价格，导致输入型通胀压力，抑制股市估值。 |

**典型市场关联性**
- BDTI 与 **布伦特原油价格（Brent Crude）** 高度相关，但领先 1-2 个月；
- 与 **波罗的海干散货指数（BDI）** 共同反映国际贸易活跃度；
- 与 **全球制造业 PMI** 呈正相关关系；
- 高波动时期常见于地缘政治事件（如中东冲突、俄乌战争）或能源结构性短缺阶段。

**历史走势特征**
- 2004-2008 年：受原油需求与海运繁荣推动，BDTI 创出历史高点（>3000点）；  
- 2015-2020 年：全球原油过剩与运输船舶供给增加，指数回落；  
- 2022 年：俄乌冲突爆发，全球能源运输重构，BDTI 一度暴涨。  

**与股市关系总结**
- 🚢 当 BDTI 上升：航运、造船、油服、能源产业链板块可能领涨；  
- 🏦 当 BDTI 下行：反映全球需求趋弱，周期股和出口导向型企业可能承压；  
- ⚠️ 若伴随油价上涨，则需警惕通胀上行与央行紧缩风险。  

---

**结论**  
原油运输指数（BDTI）是监测全球能源物流链和经济活跃度的关键指标，  
既能作为**国际贸易与航运周期**的风向标，也能为投资者提供**能源与通胀预判信号**。  
其与A股中**航运、造船、油服、海工装备**等板块联动性最强。



### 2️⃣ 行业信息（Industry-Level Factors）

“选对赛道比努力更重要”。
行业走势往往决定企业成长空间。

| 维度         | 分析要点                      | 投资启示                      |
| ---------- | ------------------------- | ------------------------- |
| **行业生命周期** | 初创期 → 成长期 → 成熟期 → 衰退期     | 成长期行业（如新能源、AI、半导体）具备更高成长性 |
| **政策导向**   | 政府支持（补贴、税收优惠）或限制（环保、监管）   | 跟随国家战略方向投资（如能源转型、国产替代）    |
| **产业链格局**  | 上游成本 → 中游制造 → 下游消费        | 找到利润高、定价权强的环节             |
| **竞争格局**   | 垄断/寡头 → 稳定利润；充分竞争 → 利润波动大 | 优选龙头公司（市占率高、议价能力强）        |

### 3️⃣ 公司信息（Company-Level Factors）

个股研究的核心是判断“公司是否值得长期持有”。

| 分析维度      | 关键指标            | 投资判断             |
| --------- | --------------- | ---------------- |
| **盈利能力**  | 营收增长率、净利率、ROE   | 持续高ROE表示企业具备竞争壁垒 |
| **偿债能力**  | 资产负债率、现金流量      | 财务稳健、负债合理的公司更抗风险 |
| **成长能力**  | EPS增速、经营现金流     | 增长稳定的公司适合中长期投资   |
| **估值水平**  | PE、PB、PEG、DCF估值 | 在低估时买入可提高长期回报    |
| **管理层质量** | 诚信、执行力、战略眼光     | 关注股东大会纪要、公开采访等资料 |

### 4️⃣ 市场信息（Market-Level Factors）

市场信息体现的是“情绪”与“节奏”。

| 信息类型        | 含义                      | 对股市的影响                  | 投资应对      |
| ----------- | ----------------------- | ----------------------- | --------- |
| **市场情绪**    | 贪婪或恐惧情绪                 | 极度乐观 → 高位风险；极度悲观 → 低位机会 | 逆向投资      |
| **资金流向**    | 主力资金、外资（北向资金）           | 资金流入 → 市场看多；资金流出 → 市场看空 | 跟随大资金     |
| **交易量与波动率** | 放量上涨 → 多头强势；缩量下跌 → 市场平稳 | 配合价格趋势判断买卖时机            |           |
| **技术形态**    | 均线系统、支撑/压力位、突破形态        | 辅助判断进出场点                | 用作短线或择时参考 |

### 1️⃣ 建立清晰的投资哲学

### 🧠 技术分析模块示例

常用技术指标计算：

```python
import pandas as pd

def moving_average(data, window=20):
    """移动平均线 (MA)"""
    data[f'MA{window}'] = data['close'].rolling(window).mean()
    return data

def macd(data, short=12, long=26, signal=9):
    """MACD 指标"""
    data['EMA_short'] = data['close'].ewm(span=short, adjust=False).mean()
    data['EMA_long'] = data['close'].ewm(span=long, adjust=False).mean()
    data['MACD'] = data['EMA_short'] - data['EMA_long']
    data['Signal'] = data['MACD'].ewm(span=signal, adjust=False).mean()
    return data
```
