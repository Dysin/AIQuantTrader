## marco_data.py

### 🔹 美国宏观指标 mapping（FRED / Yahoo Finance fallback）

| 指标名      | 描述         | 数据源   | Yahoo 代码 | 对股市潜在影响                         |
| -------- | ---------- | ----- | -------- | ------------------------------- |
| GDP      | 国内生产总值     | FRED  | ^GSPC    | GDP 增长加快 → 经济繁荣 → 企业盈利增加 → 股市上涨 |
| CPI      | 消费者价格指数    | FRED  | ^IRX     | 通胀过高 → 加息 → 股市承压；适度通胀 → 企业利润增加  |
| FEDFUNDS | 联邦基金利率     | FRED  | ^IRX     | 利率上升 → 债券更有吸引力 → 股市可能下跌         |
| UNRATE   | 失业率        | FRED  | ^TNX     | 失业率高 → 消费下降 → 企业盈利下滑 → 股市承压     |
| PCE      | 个人消费支出价格指数 | FRED  | ^CPI     | 通胀指标 → 影响美联储政策 → 股市波动           |
| M2       | 货币供应量      | FRED  | ^M2SL    | 货币宽松 → 股市上涨；紧缩 → 股市承压           |
| SP500    | 标普500指数    | Yahoo | ^GSPC    | 反映股市整体趋势，技术分析常用                 |
| NASDAQ   | 纳斯达克指数     | Yahoo | ^IXIC    | 科技股指数，趋势投资参考                    |

### 🔹 中国宏观指标 mapping（Tushare / Yahoo Finance fallback）

| 指标名    | 描述            | 数据源     | Yahoo 代码  | 对股市潜在影响                                |
| ------ | ------------- | ------- | --------- | -------------------------------------- |
| GDP    | 国内生产总值（年度）    | Tushare | 000001.SS | GDP 增长加快 → 经济复苏/繁荣 → 企业盈利增加 → 股市上涨     |
| CPI    | 居民消费价格指数（月度）  | Tushare | 000001.SS | CPI 高 → 通胀压力 → 政策收紧 → 股市可能承压           |
| M2     | 广义货币供应量（月度）   | Tushare | CNY=X     | M2 增长 → 流动性宽松 → 股市上涨；下降 → 股市承压         |
| PMI    | 制造业采购经理指数（月度） | Tushare | 399001.SZ | PMI >50 → 制造业扩张 → 股市活跃；<50 → 收缩 → 股市承压 |
| SHIBOR | 上海银行间拆借利率     | Tushare | ^CNYIR    | 利率上升 → 股市承压；下降 → 股市上涨                  |
| 失业率    | 城镇调查失业率       | Tushare | 000001.SS | 失业率上升 → 消费下降 → 股市压力                    |
| 财政支出   | 财政支出总额        | Tushare | 000001.SS | 财政扩张 → 经济刺激 → 股市利好                     |
| 上证指数   | 上证综指          | Yahoo   | 000001.SS | 股市整体趋势，技术分析参考                          |
| 深证成指   | 深证成指          | Yahoo   | 399001.SZ | 制造业和科技类股趋势参考                           |
| 人民币汇率  | CNY/USD       | Yahoo   | CNY=X     | 人民币升值 → 外资流入 → 股市上涨；贬值 → 资本外流          |
